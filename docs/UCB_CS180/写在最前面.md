# 写在最前面

> Final Grade: A

这是UCB的经典课程。课程内容深入浅出，不局限于现代‘大力出奇迹’式的深度学习神经网络方式，也目光放在了一些传统的计算机视觉算法和技巧上。除了视觉，还涉及了NeRF(Berkeley老传统)，以及VAE，Diffusion和Flow Model这些生成式模型，与时俱进。令人意外的是，CNN相关设计的很少。

课程的Project有点折磨。最有意思的是写网站，老师有意培养学生们宣传自身工作的习惯，这其实也是很重要的一个习惯。考试内容中规中矩，没有过多聚焦于数学推导而是考察知识和概念的理解和运用，难度适中。Cheatsheet是期中一张期末两张。
